# Ansible-Based Linux Server Automation Framework

## Overview
Automated Linux server administration using Ansible across multiple RHEL nodes.

## Features
- Package automation
- User management
- Service management
- File deployment
- Variables and facts
- Loops and conditions
- Multi-node administration

## Technologies
- RHEL 10
- Ansible
- YAML
- SSH

## Playbooks
- install_packages.yml
- user_management.yml
- service_management.yml
- file_deployment.yml
- variables.yml
- loops.yml
- facts.yml
